Задача состоит из двух основных подзадач:

Подзадача 1: Парсинг данных со страницы "А"
1. На веб-странице А необходимо выполнить парсинг HTML-кода и построчно извлечь данные о финансовых инструментах, представленных в виде таблицы (около 200 строк).
2. Все строки таблицы, содержащие информацию по каждому финансовому инструменту, нужно сохранить в CSV-файл "Ф". Формат файла: каждая строка таблицы записывается в CSV, а также включаются ссылки на страницы "Б", соответствующие каждому инструменту.
3. Для каждой уникальной ссылки на страницу "Б" требуется: Открыть страницу "Б". Извлечь из HTML-кода страницы определённые данные (пара ключевых показателей), соответствующие строке финансового инструмента на странице "А". (см. id="performance" - class="table-holder" - Sharpe Ratio) Добавить эти данные в соответствующую строку файла "Ф".
4. Итоговый файл "Ф" должен сохраняться с уникальным именем, основанным на текущей дате, и будет использоваться для ручной сортировки.


Подзадача 2: Извлечение котировок со страниц "Б"
1. На каждой странице "Б" также содержатся данные о котировках финансового инструмента, представленные в формате "Дата: Цена". (см. id="performanceLineChart - var productDataSets - dataProvider")
2. Эти котировки нужно считать, отформатировать и записать в отдельные файлы "P_x", где каждая строка будет представлять данные в формате: "Дата, Цена"
3. Файлы следует именовать как номер инструмента_Название инструмента_Текущая дата.csv, создавая один файл для каждого инструмента.
4. Названия всех инструментов для парсинга берутся со страницы "А".

Примеры:

Пример страницы А: https://volkswohl.tools.factsheetslive.com/
Пример страницы Б: https://volkswohl.tools.factsheetslive.com/produkt/DE0009752253/1/
Пример структуры данных в файле P_x: DD/MM/YYYY, Price


Требования к реализации
1. Я рекомендую использовать язык программирования Python и библиотеку BeautifulSoup для парсинга HTML.
2. Код должен быть прокомментирован так, чтобы был понятен человеку с начальным уровнем программирования.
3. страницы представлены на немецком языке, но это не должно быть проблемой. Все названия можно перенять 1:1.
4. Команды прошу не откликаться, так как задача довольно простая
